# Product Overview

IAM Dashboard is a cybersecurity operations platform — a "security team work haven" — that centralizes AWS security findings, vulnerability scanning, and compliance tracking into a single pane of glass.

## Target Users
- SOC Analysts: triage and track findings across IAM, EC2, S3, Security Hub, GuardDuty, Inspector, and Macie
- Security Engineers: compliance scoring, risk assessment, and remediation ticket management

## Core Capabilities
- Real-time AWS security posture across multiple services
- Case/ticket management for security findings
- Compliance tracking (SOC2, PCI-DSS, HIPAA)
- Grafana/Prometheus monitoring integration
- DevSecOps scanning pipeline (OPA, Checkov, Gitleaks)
- AWS Lambda-based security scanner with API Gateway

## Key Principle
Reduce tool-switching for security teams by consolidating scanning, triage, ticketing, and reporting in one place.
